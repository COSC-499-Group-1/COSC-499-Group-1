Markdown Guide

The message box in which you type messages in supports a wide verity of markdown options, for you to add style and flair to your messages. Here is a run down of all of the options available to you

Keyboard Usage

There are a couple useful keyboard shortcuts to know:

  • Enter: Press enter anytime to send a message.
  • Shift+Enter: Press shift enter to enter a new line into your text. (Note: multiple consecutive newlines that that are not part of a code block will be collapsed into one new line.)
  • @ key: press the '@' key anytime to start "mentioning" a user. See mentioning a user list
  • Up and down arrow keys: While mentioning, use the up and down arrow keys to select and highlight a user to mention. Press the enter key to select the user you've selected.

Lists

In this example, leading spaces are shown with dots (⋅)

        1. First ordered list item

        2. Another item

        ⋅⋅* Unordered sub-list.

        1. Actual numbers don't matter, just that it's a number

        ⋅⋅1. Ordered sub-list

        4. And another item.

        * Unordered list can use asterisks

        - or minuses

        + or pluses

produces...
